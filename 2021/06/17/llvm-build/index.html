llvm-build

在macos上编译llvm,使用ninja来编译。

克隆仓库
‘’‘
git clone https://github.com/llvm/llvm-project
’‘’

设置编译选项
‘’‘
mkdir build
cd build
cmake -G Ninja -DCMAKE_BUILD_TYPE=Debug -DLLVM_USE_SPLIT_DWARF=ON -DCMAKE_INSTALL_PREFIX=/Users/bytedance/llvm-product -DCMAKE_C_COMPILER=/usr/bin/clang -DCMAKE_CXX_COMPILER=/usr/bin/clang++ -DLLVM_ENABLE_PROJECTS=”clang;clang-tools-extra;lld;lldb;libcxx” -DLLVM_TARGETS_TO_BUILD=”X86” -DLLVM_USE_NEWPM=ON ../llvm
’‘’

编译
‘’‘
ninja -j16
‘’‘

安装
‘’‘
ninja install
’‘’
